Liam Roadtrip - Godot 4 Project

How to run:
1. Download and unzip this project.
2. Open Godot 4, choose 'Import' and point to the project.godot file or 'Open'.
3. Open the Main.tscn scene and run.

Controls:
- Persuasion phase: click choices with mouse.
- Driving phase: arrow keys (Up/Down accelerate/brake, Left/Right steer).

Notes:
- Station names are set at runtime by Driving.gd: the first station encountered will be '7/11' and the second 'QuikTrip'.
- This is a minimal prototype; feel free to expand art, collision, audio, and polish.
